- **Style:** Modern with rounded edges

### Logo
- Use your app icon (icon-192.png) in the center
- Make sure it doesn't cover more than 30% of the QR code

### Size
- **For digital use:** 300x300 pixels minimum
- **For print:** 1000x1000 pixels or vector (SVG)
- **Rule of thumb:** Bigger is better for scanning
